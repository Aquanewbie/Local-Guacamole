## Local-Guacamole ##
## Summary ##
A Visualization Mapping the Production of Avocado and the additional ingredients in Guacamole.

## Methods: ##
- <br/>
-Rows with Null values for gender are dropped. <br/>
-All visualizations are plotted with colors utilized in HTML and CSS. <br/>
-All visualizations are plotted with transparent backgrounds. <br/>
